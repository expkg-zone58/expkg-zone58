A collection of XQuery libraries targeted at BaseX, packaged using EXPath package format http://expath.org/

Often these are wrappers to third party open-source Java libraries.

ex-xparse
Parse expressions in XPath or XQuery into XML trees using REx generated parsers
mp3tag-extract
Extract metadata from mp3 format files using the jaudiotagger library.
ex-thumbnailator
An XQuery interface to the thumbnail generator library [thumbnailator]
ex-schematron
An XQuery packaging of ISO Schematron for BaseX
metadata-extract
An XQuery package to extract image metadata as XML using the Drew Noakes library
ex-graphviz
XQuery interface for Graphviz with EXPath packaging.
ex-dotml
The dotML library provides conversion from dotML format to graphviz dot
